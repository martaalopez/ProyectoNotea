{{"Acerca de Notea" }}

Esta es una página web desarrollada en Angular con el objetivo de aprender a desarrollar aplicaciones web con este framework. También se usa Firebase como base de datos para guardar las notas. Puedes iniciar sesión con tu cuenta de Google para aprender a crear tus notas. Podrás crear notas, eliminarlas, editarlas y, además, gracias a tener iniciada sesión con Google, podrás descargar la app en tu teléfono y tener las notas sincronizadas.
